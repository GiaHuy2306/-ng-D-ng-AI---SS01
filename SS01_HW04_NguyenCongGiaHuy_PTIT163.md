# BÀI 4: Phân tích & Lựa chọn

## Tối ưu hóa hiệu năng

### Đáp án lựa chọn: Prompt 2

Em chọn **Prompt 2** vì prompt này có mục tiêu và yêu cầu rõ ràng hơn. Prompt 2 nói cụ thể rằng cần chuyển hàm Java đang dùng vòng lặp `for` sang **Java Stream API**, đồng thời yêu cầu code ngắn gọn, dễ bảo trì và có giải thích lý do tối ưu.

Prompt 1 chưa tốt vì câu “tối ưu hóa code Java này giúp tôi chạy nhanh hơn” còn quá chung chung. AI có thể hiểu sai mục tiêu là tối ưu tốc độ xử lý, trong khi yêu cầu chính của bài là làm cho code ngắn gọn và dễ bảo trì hơn. Prompt này cũng không đưa ra ràng buộc cụ thể như dùng Stream API hay cần giải thích kết quả.
